Documentation Overview

Here is a breakdown of the key concepts and integrations available in the Ellevux platform.

Getting Started

Authentication

Learn how to securely log in, or for developers, how to generate API Keys so you can securely connect your own custom software to Ellevux without exposing your main credentials.

Call Routing

Instructions on how to tell the AI what to do with different types of calls. (e.g., "If the caller says emergency, forward the call to my cell phone. If they want a quote, send them a pricing text.")

Integrations

Connect Ellevux to the software tools you are already using to run your business.

Google Calendar

Connect your business calendar so the AI knows exactly when agents are available and can book appointments directly, entirely avoiding double-booking.

HubSpot

HubSpot is one of the most popular CRM platforms for local businesses. This integration allows Ellevux to automatically log call transcripts, check client histories, or insert new leads directly into your pipeline without manual data entry.

Webhooks

A technical feature for advanced users. A webhook is a way for Ellevux to instantly send data to another app. For example, a webhook could be set up to instantly ping a Slack channel with a call summary the moment a customer hangs up the phone.
